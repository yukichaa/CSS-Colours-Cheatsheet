How to Use CSS Colors

Element Example Description
Color
 color: red; Changes the color of foreground elements, such as the text itself. There are 140 named colors in CSS, e.g. red, blue, aqua etc.
Background Color background-color: yellow; Alters color of background elements, providing a colored backdrop to the content.
Hex
 color: #2345AB; A way of representing color. First two digits (each hexadecimal digit ranging from 0 to F) denote red value, while second two represent green, and last two represent blue.
RGB
 color: rgb(140, 12, 200); Each number simply represents the prevalence of red, green and blue respectively from 1 to 255.
HSL
 color: hsl(178, 70%, 30%); Stands for Hue, Saturation, Lightness. First number represents a hue degree on the colour wheel. Second number is the saturation from 0 to 100% and third number is lightness from 0 to 100%.
Alpha
 color: rgba(140, 12, 200, 0.5);
&
color: hsla(178, 70%, 30%, 0.5);		 Alpha can be appended onto rgb or hsl representations of color to denote opacity. A value of 0 will be fully transparent, while an alpha value of 1 is fully opaque.
